# MY LIFE IS SOCCER - Version Statique HTML/CSS

## Description
Ceci est une version statique HTML/CSS de votre site MY LIFE IS SOCCER. Elle peut être hébergée sur n'importe quel serveur web traditionnel, y compris WordPress.

## Contenu du dossier
- `index.html` - Page principale du site
- `assets/` - Fichiers CSS et JavaScript compilés
- Images, vidéos et autres ressources

## Installation sur WordPress

### Option 1 : Importer comme thème personnalisé
1. Téléchargez les fichiers statiques
2. Créez un dossier `football2enjoy` dans `/wp-content/themes/`
3. Copiez les fichiers du dossier `dist/public/` dans ce dossier
4. Créez un fichier `style.css` à la racine du thème avec :
```css
/*
Theme Name: MY LIFE IS SOCCER
Theme URI: https://mylifeissoccer.nl
Description: Site de formation de football
Version: 1.0
*/
```
5. Activez le thème dans WordPress

### Option 2 : Hébergement traditionnel (recommandé)
1. Téléchargez les fichiers statiques
2. Connectez-vous à votre serveur FTP/SFTP
3. Uploadez tous les fichiers à la racine de votre domaine
4. Accédez à votre site via le navigateur

## Fonctionnalités
- ✅ Page d'accueil avec vidéo
- ✅ 3 formations (4-8 ans, 9-12 ans, 13+)
- ✅ 6 profils d'entraîneurs
- ✅ Design responsive Cogni:Wave
- ✅ Formulaires d'inscription
- ✅ Section contact

## Notes importantes
- Les formulaires d'inscription ne fonctionnent pas en version statique
- Pour les formulaires, vous devrez ajouter un service externe (Formspree, Netlify Forms, etc.)
- Les vidéos .mov ne sont pas incluses (utilisez MP4 à la place)
